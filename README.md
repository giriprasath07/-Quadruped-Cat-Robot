# 🐱 Quadruped Dog Robot

A compact and lightweight **Quadruped Cat Robot** designed using **3D-printed parts**, **servo motors**, and a **microcontroller**. This project demonstrates four-legged locomotion, servo synchronization, and robotic movement while maintaining a modular design that is easy to modify and upgrade.

The robot is ideal for robotics enthusiasts, students, and researchers interested in quadruped robots, embedded systems, and motion control.

## 🌟 Project Goal

The goal of this project is to build an affordable, open-source quadruped robot that helps students and hobbyists learn robotics, embedded systems, servo control, and robot locomotion. The modular architecture makes it easy to add advanced capabilities such as computer vision, autonomous navigation, and AI-based behaviors.
